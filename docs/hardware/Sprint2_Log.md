**MS200K Lidar** (oradar)
-Measurement Range: 0.03 to 12.0m @90% reflectivity
-Data Contents: Distance, angle, intensity, timestamp
-FOV(Horizontal): 360 Degrees
-Measure Frequency: 4500Points/s
-Frame rate: 7Hz to15Hz, default 10Hz
-Laser wavelength: 905nm
-Voltage supply: 5V DC
-Peak power: 2.5W
_4 PIN Connector_ :
             1= TX (serial data sender)
             2 = RX (Serial data receiver)
             3 = GND
             4 = VCC
_Serial Port Configuration_:
             Baud rate(bps) = 230400
             Data bit = 8
             Stop bit = 1
             Check bit = None
             Flow Control = None
_Working Mode_:
              Ranging mode: Successfully initialized and normal ranging operation.
              Standby mode: Successfully initialized but no laser beam is emitted, no data transmission.
